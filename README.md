# UniRent 🎓🛒

**UniRent** is a premium, peer-to-peer rental marketplace designed specifically for academic communities. It enables students to safely list, discover, and rent items to one another within a trusted campus ecosystem.

### ✨ Key Features
- **Student Stores:** Users can set up verified personal storefronts to list their rental items.
- **Secure Bookings:** A complete system for managing rental requests and seamless transactions.
- **Real-time Chat:** Built-in messaging to facilitate communication between renters and owners.
- **Trust & Verification:** Admin verification and a community review system ensure platform safety.

### 💻 Tech Stack
- **Frontend:** Next.js (App Router), React, Tailwind CSS 
- **Backend:** Node.js, Express.js, MongoDB (Mongoose)
- **State Management:** Redux Toolkit
